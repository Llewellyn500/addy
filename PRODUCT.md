# Addy Product Context

## Register

Product

## Purpose

Addy is a small cross-platform network utility that shows active local network interfaces, connection labels, IPv4 addresses, and IPv6 addresses with fast copy actions.

## Users

People who regularly need to grab local IP addresses while developing, testing, troubleshooting, or moving between networks.

## Product Principles

- Keep the app instantly readable and quick to operate.
- Make copying an address the primary action on every platform.
- Preserve the same Addy identity across desktop and Android, even when platform behavior differs.
- Avoid idle work; the app should feel lightweight and responsive.

## Brand And UI Direction

Addy uses a dark violet-black canvas, vivid violet leaf-shaped panels, hard black outlines and offset shadows, white text, lavender secondary text, and a yellow action accent. The interface should feel compact, confident, and task-focused rather than decorative.

## Platform Notes

Desktop Addy runs as a system tray utility. Android cannot use desktop-style tray behavior, so the mobile app presents the same network-address copy workflow as a foreground companion app.

## Anti-References

- Generic Material cards that lose the Addy leaf-card identity.
- Marketing-style screens or onboarding before the actual network list.
- Dense diagnostics that compete with the copy workflow.
- Soft glassy surfaces, gradients, or low-contrast muted text.
