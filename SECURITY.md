# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| latest  | ✅ Yes             |
| older   | ❌ No              |

Only the latest release is supported with security updates.

## Reporting a Vulnerability

If you discover a security vulnerability, **please do not open a public issue.**

Instead, email **llewellynpaintsil34@gmail.com** with:

- A description of the vulnerability
- Steps to reproduce it
- The potential impact

You should receive a response within **72 hours**. If confirmed, a fix will be released as soon as possible and you will be credited (unless you prefer to remain anonymous).

## Scope

Addy is a local-only utility that reads network interface information from your operating system. It does **not**:

- Make any network requests
- Collect or transmit any data
- Require elevated / administrator privileges
